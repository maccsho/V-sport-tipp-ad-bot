## Projekt

Egyszerű statikus web projekt váz.

### Futtatás

1. Nyisd meg a `index.html` fájlt böngészőben.
   - Linux/macOS: `xdg-open /workspace/project/index.html` vagy `open /workspace/project/index.html`
   - Windows: dupla kattintás vagy `start /workspace/project/index.html`

### Struktúra

```
project/
├─ index.html
└─ src/
   ├─ main.js
   └─ styles.css
```

### Testreszabás

- A stílusokat a `src/styles.css` fájlban módosíthatod.
- A működést a `src/main.js` fájlban bővítheted.
